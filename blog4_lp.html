Make your first contribution!

First contributions are always special! It may be as small as a typo change but it is your first step into this exciting world of Open Source! This blog is aimed to walk you through the steps involved in making your first pull request!
If you are a complete beginner, check this workshop conducted by WnCC or you can follow this tutorial.
If you are aware of the basics, follow along to learn how to make your first pull request!

#1: Choose the project

Check the projects under Hello FOSS repository and the instructions given in the repository of your choice.

#2: Fork the repository

Fork the repository of your choice by clicking on the fork button on the top of the github page of the respective repository. This step will create a copy of the repo in your account.

#3: Clone the repository

Now clone the forked repository to your machine. Go to your GitHub account, open the forked repository, click on the Code button and then click the copy to clipboard icon.

Open a terminal and run the following git command: git clone "url" where "url" is the url you copied.
For example: git clone https://github.com/username/Hello-FOSS.git where username is your GitHub username. The contents of your copy of the project are now copied to your machine.

#4: Create a branch

Change to the repository directory on your machine: cd Hello-FOSS
Create a branch using the command: git checkout -b new-branch-name

#5: Make necessary changes and Commit

Now complete the task in this branch. Make all necessary changes here. Now commit these changes using the following commands.
git add .
git commit -m "Commit message"

#6: Push changes to Github

Push your changes using the command : git push origin new-branch-name This step will save all your local changes to your forked repository on GitHub in a branch "new-branch-name".

#7: Hit Us a pull request

Go to your repository on GitHub and you will see a Compare and Pull Request button. Click on that and then submit the pull request. Ensure that you follow this template while sending your Pull Request.
