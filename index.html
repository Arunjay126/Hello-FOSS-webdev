All you need to know

  • What is Hello-FOSS?

    Festival launched by WnCC, IIT Bombay to instill and encourage the culture of Open Source in the institute

  • Purpose of this Website

    If you were stuck with some error and you feel someone else can fall into this too, add a blog here regarding issue and it's fix cause CODERS TOGETHER STRONG! :)

  • Fields of Projects

    To cover interests of most of the crowd, we have projects on DSA, Web and App Developement and, Machine Learning

  • Get Started

    Links to the projects have been added below, go to the repos, read the instructions there, make your changes to the repo and create a pull request and you're done!

  • Participation in Hacktoberfest

    All the projects have repositories under the hacktoberfest tag, so contributing to this project gets you participation in Hacktoberfest as well. PS: Don't forget to register on the Hacktoberfest website

  • Discussion Forum

    Join the Discord server to get connected to the community. Feel free to ask for help in case you're stuck or clueless at any point
