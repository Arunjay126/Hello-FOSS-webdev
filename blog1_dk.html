General Instructions to start this repository on your local host and to start making changes.

  1. Install Jekyll on your system using this link. You can refer other tutorials as well.
  2. Clone the repository in your system and navigate to it using CLI and run command jekyll serve on your computer. Then the terminal says that the website has been launched on your localhost. Open your localhost and you should see the website there. If you face any errors, google them and most will get fixed, if you still face an error, contact on the discord server.
  3. Refer to this website to get some basic idea about what is Jekyll. You don't have to go through all the tutorials, just sufficient enough such that you are able to understand the contents of this cloned folder.
  4. You are good to go and start contributing!
  5. Most syntax that you have to use/edit is HTML, thus making it easier to build beautiful websites with minimal effort.
